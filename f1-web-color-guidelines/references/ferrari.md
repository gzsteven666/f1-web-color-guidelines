# 法拉利车队风格的 HTML 设计指引

## 设计目标

做出 **高辨识度、高激情、精致且带有意式赛车传统感** 的网页视觉。
整体应接近 Scuderia Ferrari 近年的公开视觉：**更亮、更强烈的 Rosso Scuderia 为绝对主识别，白色在 2026 开始承担更明确的结构角色，黄色只作为血统点缀，深色与炭黑用于压重心和提升高级感**。如果按 2026 官方语义去理解，这套设计的关键变化不再只是“更深的红”，而是 **更 bright and intense 的 Rosso Scuderia、white 在 cockpit 与 engine cover 周围承担新的结构角色，以及 gloss paint 的回归**。这让 Ferrari 的网页风格也不应只是“深红 + 黄点缀”，而要更接近 **高张力红白结构 + 高光泽赛车漆面** 的组合，同时依然保留意式赛车传统与攻击性。

## 整体气质关键词

* italian racing passion
* rosso scuderia intensity
* premium speed
* dramatic contrast
* motorsport elegance
* sculpted aggression
* gloss paint heritage
* red-white structural contrast

## 一、核心配色方案

### 1) 品牌核心层

这层负责让页面一眼有 Ferrari F1 的味道。

* **主红 / Rosso Scuderia Base**：`#D3141C`
* **深红 / Rosso Deep**：`#A3161B`
* **辅助深红 / Rosso Shadow**：`#6D0F13`
* **结构白 / Ferrari White**：`#F4F2EE`
* **炭黑 / Carbon Black**：`#111111`
* **法拉利黄 / Giallo Accent**：`#F7D117`

这里的网页复现逻辑是：以 Ferrari 官方 2026 明确强调的 **更亮、更强烈的 Rosso Scuderia** 为基础，同时把白色从“普通留白”提升为 **结构性角色**。也就是说，白色不再只是文字色或赞助留白，而是可以承担 **分割、速度带、车身结构切线** 的功能；黄色则继续保留，但只作为 Ferrari 品牌血统的微量强调，而不再是明显的第二主色。

### 2) 推荐可替代值

如果你希望网页更接近“意大利高性能机械 + 高光泽赛车烤漆”的感觉，可以在不偏离车队识别的前提下，用下面这组替代值：

* **更亮主红**：`#D81E26`
* **高光红 / Rosso Highlight**：`#F04A4A`
* **深酒红 / Bodywork Shadow**：`#7A1116`
* **结构暖白 / Ivory White**：`#F7F4F1`
* **深炭黑**：`#0E0E0F`
* **血统黄强化版**：`#FFD230`

这组值更适合做 dashboard、图表、主视觉层级，也更容易做出 **法拉利 2026 那种高对比、光泽感更强、红白关系更明确** 的视觉效果。

### 3) 页面结构层

* **主背景**：`#140B0C`
* **一级卡片底**：`#1A1012`
* **二级卡片底**：`#201214`
* **边框 / 分割线**：`#4A262A`
* **主文字**：`#F7F4F1`
* **次文字**：`#D7CDCA`
* **弱文字**：`#A69390`

### 4) 数据可视化层

法拉利适合用 **Rosso Scuderia 主数据 + 更亮红高光 + 白色结构对比 + 极少量黄色提示** 的方式做图表，不适合引入过多冷色。

* **Series 1**：`#D3141C`
* **Series 2**：`#F04A4A`
* **Series 3**：`#A3161B`
* **Structure**：`#F4F2EE`
* **Accent / Warning**：`#F7D117`
* **Neutral**：`#8F7A77`

## 二、网页配色占比

法拉利这套最关键的是：
**用 Rosso Scuderia 建立主身份，用更亮的红制造速度感和受光层次，用白色建立结构对比，用深色压住体积，用极少量黄色保留血统点缀。**

### 推荐整页占比

* **深红 / 炭黑背景体系：42%**
* **Rosso Scuderia 主识别：24%**
* **更亮红高光层：10%**
* **白色结构对比：10%**
* **文字与浅中性色：12%**
* **法拉利黄点缀：2%**

这个比例更接近 2026 Ferrari 的真实视觉方向：红色依旧是主角，但它不再只是“深色烤漆里的厚重”，而是加入了更明确的 **高光泽、红白结构对比**。黄色必须进一步克制，更多是血统标记而不是第二主色；白色则开始承担更明确的结构角色，但依然不能做成大面积白底。

### dashboard 页面推荐占比

* 页面底色：`#140B0C`，约 **54%**
* 卡片 / 面板：`#1A1012` / `#201214`，约 **20%**
* 主红识别块 / 主图表：约 **14%**
* 白色结构对比与关键数值：约 **10%**
* 黄色提示与血统点缀：约 **2%**

### 强限制

* 不要把黄色做成大面积主色
* 不要把页面做成普通红白促销风
* 黄色总面积建议 **不超过 2%**
* 页面必须保持“红色是主体、深色是体积、白色是结构、黄色是血统点缀”的关系
* 不要把 Ferrari 红做成偏粉或偏橘的互联网红
* 不要让整体失去 **意式高性能与赛车传统感**

## 三、元素级配色规范

### 背景

* `body`：`#140B0C`
* section 背景：`#180E10`
* card 背景：`#201214`
* modal / overlay：`#2A1719`

背景必须偏深，像 **高光泽红色赛车车漆在维修区灯光下的暗部层次**，而不是纯黑后台。

### 导航栏

* 背景：`rgba(20, 11, 12, 0.90)`
* 边框下沿：`1px solid #3A2323`
* Logo / 标题高亮：`#F7F4F1`
* 当前激活项：`#D3141C`
* 非激活项：`#B9AAA6`

导航应体现 **高辨识、克制、贵、快**，不要做成电商促销红条。

### 卡片

* 背景：`#201214`
* 边框：`1px solid #4A262A`
* 标题：`#F7F4F1`
* 正文：`#D7CDCA`
* 辅助说明：`#A69390`
* 顶部或左侧可加 `2px` 的 `#D3141C` 细线作为状态高亮
* 关键卡片可加入 `#F4F2EE` 的结构切线，模拟 2026 车身上的白色结构角色
* `#F7D117` 只允许极少量点缀，而且只能点到为止

如果要更像法拉利车身，可以在大卡片背景里加入非常轻微的 **深酒红 → Rosso Scuderia → 局部高光红 / 结构白** 过渡，模拟赛车烤漆与车身切线在曲面上的层次。

### 按钮

#### 主按钮

* 背景：`#D3141C`
* 文字：`#FFFFFF`
* Hover：`#E32832`
* Active：`#A3161B`

主按钮应该像 **车身编号区或赛车主涂装高识别区**，醒目、直接、有压迫感。

#### 次按钮

* 背景：transparent
* 边框：`1px solid #8C7C79`
* 文字：`#EFE7E4`
* Hover 背景：`rgba(211, 20, 28, 0.10)`

### 数据图表

* 主线 / 主柱：`#D3141C`
* 第二序列：`#F04A4A`
* 第三序列：`#A3161B`
* 结构对比：`#F4F2EE`
* 提示 / 峰值：`#F7D117`
* 中性对比：`#8F7A77`
* 坐标文字：`#B4A4A0`
* 网格线：`rgba(255,255,255,0.08)`

图表核心应是 **激情、清晰、强对比**。白色可以承担结构性对比，黄色只用来做提醒、峰值、排名变化，不要泛滥。

### 文字

* 一级标题：`#F7F4F1`
* 二级标题：`#E7DCD8`
* 正文：`#D7CDCA`
* 弱化说明：`#A69390`

## 四、最像“法拉利车身”的网页渐变

法拉利网页里最该复现的不是单一鲜红，而是 **Rosso Scuderia 车漆底 + 受光亮红 + 白色结构切线 + 极少量黄色血统点** 的关系。2026 官方语义的关键不是“更多元素”，而是 **更亮、更强烈的红与更清楚的结构对比**，这也更适合网页里的高级感。

### 方案 A：最像赛车主车身的主视觉

```css
background: linear-gradient(
  135deg,
  #140B0C 0%,
  #6D0F13 32%,
  #D3141C 74%,
  #F4F2EE 100%
);
```

这个版本适合首页 hero、赛车主题 banner 或品牌主视觉，重点是像 **高光泽 Ferrari 车身被主光扫过，同时在结构边缘露出白色切线**。

### 方案 B：更稳的 dashboard 版

```css
background: linear-gradient(
  135deg,
  #180E10 0%,
  #201214 38%,
  #A3161B 72%,
  #D3141C 100%
);
```

这个版本更适合后台、数据面板、比赛数据页，整体更稳，不会太像宣传海报。

### 方案 C：更强调意式高性能与结构对比的版

```css
background: linear-gradient(
  120deg,
  #140B0C 0%,
  #2A1719 30%,
  #A3161B 58%,
  #D3141C 84%,
  #F4F2EE 100%
);
```

这个版本更强调 **红白结构关系与高光泽漆面张力**。黄色仍然不建议进入大面积渐变终点。

## 五、网页中最容易做错的地方

* 把 Ferrari 做成普通红白电商风
* 红色太浅，失去赛车漆面的厚重与光泽
* 黄色太多，抢走主识别
* 白色用成普通大留白，失去结构角色
* 深色不够，失去体积感和高级感
* 做成促销海报，而不是赛车品牌视觉
* 只有“红”，没有 **激情、传统、机械雕塑感与 gloss paint 质感**

## 六、给 AI 的风格指令模板

```text
请将页面设计为 Scuderia Ferrari 风格的数据可视化网页。

视觉关键词：
italian racing passion, rosso scuderia intensity, premium speed, dramatic contrast, motorsport elegance, sculpted aggression, gloss paint heritage, red-white structural contrast

配色要求：
- 以深红和酒红作为主背景与主品牌识别色
- 使用 Rosso Scuderia 作为主高亮和主数据色
- 使用结构白建立分割、边线、对比信息和车身切线感
- 允许极少量 Ferrari yellow 作为血统点缀
- 整体应呈现激情、精致、意式高性能、传统赛车感
- 页面颜色关系应类似 Ferrari F1 2026 的红白结构主导，而不是普通红白促销风
- 黄色只能点到为止，不能成为第二主背景

布局要求：
- 模块层级清晰，视觉重心低，强调速度感和侵略性
- 使用细边框、轻微渐变、低对比阴影
- 可模拟赛车漆面、进气口切面和高性能机械结构的感觉
- 图表以红色层级为主，用白色做结构性对比，只用极少量黄色做提示，不要引入多余冷色

材质感要求：
- 模拟 gloss paint、motorsport bodywork、premium Italian performance finish
- 不要赛博霓虹
- 不要玻璃拟态
- 不要把黄色做成大面积主色
- 不要把页面做成普通电商红白促销风
```

## 七、最终落地判断标准

如果做对了，页面看起来应该像：

* **更亮的 Rosso Scuderia 赛车车身在灯光下有高光泽烤漆层次，并带有明确的白色结构切线**
* 不是普通红色品牌官网
* 不是促销海报
* 而是 **热、快、贵、传统、极具攻击性，并且有意大利赛车的雕塑感**

如果做错了，最常见的观感会变成：

* 红色太浅
* 黄色太多
* 白色太散
* 太像电商促销
* 没有体积感
* 没有 Ferrari 那种传统与激情并存的压迫感
