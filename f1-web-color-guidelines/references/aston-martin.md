# 阿斯顿·马丁车队风格的 HTML 设计指引

## 设计目标

做出 **深邃、克制、速度感强、带有英伦高端赛车气质** 的网页视觉。
整体应接近 Aston Martin Aramco Formula One Team 近年的公开视觉：**深色底、Aston Martin Racing Green 为主识别、偏青的高光绿建立层次、少量荧光黄绿作点缀、黑色碳纤维区域负责压重心**。官方对这套视觉的表述核心一直围绕 “Aston Martin Racing Green”，并强调它不仅是单一颜色，而是有历史、材质感和层次变化的品牌识别。按 2026 官方语义继续收紧时，这套视觉还应被理解为一种更系统化、`considered`、`deliberate` 的 works-era 工程表达，而不是单纯“豪华绿色主题”。
在网页复现时，这种绿色不应只是普通深绿色，而应更接近 **真实 F1 车身外壳上的深色金属车漆**：颜色深、表面致密、受光时有明确的线性高光，暗部厚重而干净，整体更像赛车烤漆而不是普通平涂色块。

## 一、核心配色方案

### 1) 品牌核心层

这层负责让页面一眼有 Aston Martin F1 的味道。

* **主深绿 / Racing Green Base**：`#002420`
* **高光绿 / Green Accent**：`#037A68`
* **辅助高光绿 / Secondary Accent**：`#0A7968`
* **碳黑 / Carbon Black**：`#0D0F10`
* **白色 / Signal White**：`#FFFFFF`
* **荧光黄绿 / Lime Highlight**：`#A9D23E`

其中 `#002420` 常被第三方数字色资料整理为 Aston Martin F1 Team 的深绿主色，而 `#037A68` 常见于 Aston Martin Racing 相关数字色整理，可作为网页里更亮一层的高光绿来用。需要说明的是，这些不是我能确认到的官方网页 token 原文，而是基于官方“signature Aston Martin Racing Green”方向，加上公开可用数字整理后得出的 **工程化网页复现值**。
在实际使用时，`#002420` 不应被理解成普通纯色背景绿，而更适合作为 **深色金属车漆底色**；`#037A68` 与 `#0A7968` 则更像车身曲面在摄影棚灯光下出现的受光层与反射层，而不是单纯把主色提亮一档。

### 2) 推荐可替代值

如果你希望颜色更贴近“网页中的高级车漆感”，可以在不偏离车队识别的前提下，用下面这组替代值：

* **更深背景绿**：`#011816`
* **稳定主品牌绿**：`#01352E`
* **车漆高光绿**：`#0B6F63`
* **更亮的边缘反射绿**：`#1C9B86`
* **深灰黑**：`#111315`

这组值的好处是层级更清楚，更适合 UI 系统里做 hover、图表、卡片和大面积背景渐变，也更容易做出 **金属车漆的反光逻辑**。

## 二、网页配色占比

阿斯顿·马丁 F1 这套最关键的是：
**用深绿建立身份，用黑色压住体积，用更亮的绿制造车漆反光层次，用极少量 Lime 提供速度感信号。**

### 推荐整页占比

* **深黑绿背景体系：58%**
* **深绿品牌块：22%**
* **中亮度绿强调：10%**
* **白色与浅灰文字：8%**
* **Lime 点缀：2%**

这个比例更接近官方近年视觉给人的真实观感：主色确实是 Racing Green，但整体不是一整页鲜绿，而是深底上让绿色占据识别位，再用更亮一层的绿做受光与动势。官方也明确强调 Aston Martin Racing Green 的“层次、历史与材质表达”，而不是普通平涂绿。
这里的“层次”尤其要理解成 **赛车金属漆面在不同曲面上的明暗变化**：亮部更通透，暗部更厚重，反光有方向性，而不是均匀、软塌、像 UI 渐变一样铺开。

### 强限制

* 不要把主色做成偏黄的草绿
* 不要大面积使用 Lime
* Lime 总面积建议 **不超过 3%**
* 页面必须保持“深绿是主体、黑色是体积、高光绿是反光层”的关系
* 不要把 Racing Green 做成普通平涂色块或塑料亮面
* 高光必须更像 **金属车漆受光**，而不是互联网产品常见的泛白高亮

## 三、元素级配色规范

### 背景

* 页面大背景：`#071110`
* 模块背景：`#0D1716`
* 更深一层的 section 背景：`#08100F`

大背景不要直接铺纯绿，而要偏向 **黑绿混合的深色烤漆底**，这样更像阿斯顿·马丁赛车本体的体积感。

### 卡片

* 背景：`#0D1716`
* 边框：`1px solid #1B2E2A`
* 标题：`#F3F7F6`
* 说明文字：`#A8B7B3`
* 卡片顶部或左侧可以加一条 `2px` 的 `#037A68` 细线，模拟赛车高光边线
* 如需加强车身质感，可在卡片大面积主视觉区域加入非常轻微的深绿到高光绿过渡，模拟金属漆在曲面上的反射变化，但不要做成强烈霓虹渐变

### 按钮

#### 主按钮

* 背景：`#037A68`
* 文字：`#FFFFFF`
* Hover：`#0A8B76`
* Active：`#026554`

主按钮应该像车身上的高亮识别区，偏精致、偏冷静，而不是互联网产品里那种鲜亮跳色按钮。

#### 次按钮

* 背景：transparent
* 边框：`1px solid #295C54`
* 文字：`#DCE8E5`
* Hover 背景：`rgba(3, 122, 104, 0.10)`

### 数据图表

* 主线 / 主柱：`#037A68`
* 次级数据：`#0A7968`
* 对比数据：`#A9D23E`
* 坐标文字：`#8FA7A1`
* 网格线：`rgba(255,255,255,0.08)`

图表配色一定要稳。Lime 只能作为提醒、峰值、超车、异常、激活点，不能变成第二主色。

### 文字

* 一级标题：`#F5F7F6`
* 二级标题：`#D7E1DE`
* 正文：`#AEBAB6`
* 弱化说明：`#7E908B`

文字系统应偏冷、偏克制，避免太暖的灰。

## 四、最像“阿斯顿·马丁车身”的网页渐变

官方对 Aston Martin Racing Green 的叙述重点不是“单独一个色号”，而是其在不同光线和材质上的层次变化。网页里最应该复现的是这种 **深绿底 + 偏青高光** 的逻辑。
更准确地说，应该模拟 **真实 F1 车身金属漆在摄影棚灯光下的受光方式**：高光有明确方向，亮部集中，暗部保持厚重，不是普通 UI 那种从左到右平均过渡的装饰性渐变。

### 方案 A：最像车漆高光的主视觉

```css
background: linear-gradient(
  135deg,
  #06100F 0%,
  #002420 38%,
  #037A68 78%,
  #29A892 100%
);
```

这个版本最适合做首页 hero、赛车主题 banner 或核心品牌主视觉。重点不是“更亮”，而是像 **深色金属车漆被一束主光扫过** 后形成的高光带。

### 方案 B：更稳的 dashboard 版

```css
background: linear-gradient(
  135deg,
  #071110 0%,
  #0D1716 25%,
  #002420 62%,
  #037A68 100%
);
```

这个版本更适合后台、仪表盘、数据中台页面。整体更克制，保留车漆反光感，但不会太张扬。

### 方案 C：更深、更英伦高端的版

```css
background: linear-gradient(
  120deg,
  #071110 0%,
  #0A1211 34%,
  #13211F 58%,
  #002420 82%,
  #0A7968 100%
);
```

这个版本更强调 **深色烤漆的厚重感**，适合想要更稳、更贵、更接近豪华性能品牌气质的场景。

## 五、网页中最容易做错的地方

* 把 Aston Martin 做成普通科技绿
* 绿色过亮，导致像新能源品牌而不是 F1 车队
* Lime 用太多，破坏高级感
* 黑色不够，页面失去低重心
* 渐变做成霓虹、夜店风或赛博风
* 表面高光过于平均，像 UI 装饰，而不像真实车漆反光
* 只做颜色接近，却没有做出 **金属漆面 / 碳纤维 / 车身曲面反射** 的材质层次

## 六、给 AI 的风格指令模板

```text
请将页面设计为 Aston Martin Aramco Formula One Team 风格的数据可视化网页。

视觉关键词：
british racing luxury, deep racing green, refined performance, premium motorsport, dark elegance, aerodynamic gloss, layered green depth, metallic racing paint, considered by design

配色要求：
- 以深黑绿和深碳色作为主背景
- 使用 Aston Martin Racing Green 作为主品牌识别色
- 使用偏青的高光绿作为激活态、重点图表和速度感高光
- 只允许极少量 Lime 作为提示色或信号色
- 整体应呈现深色、克制、英伦高端、赛车漆面般的质感
- 页面颜色关系应类似 Aston Martin F1 的深绿主导，而不是普通科技绿或草绿色
- 绿色必须更接近真实 F1 车身金属漆，而不是普通平涂色或塑料亮面

布局要求：
- 模块层级清晰，保持低重心和横向速度感
- 使用细边框、轻微渐变、低对比阴影
- 可模拟碳纤维和赛车漆面反光的感觉
- 图表以绿色层级为主，不要引入多余高饱和彩色
- 高光要有方向性，像曲面车身在灯光下形成的线性反射

材质感要求：
- 模拟 gloss paint、carbon fibre、premium motorsport finish
- 加入 subtle automotive metallic finish
- 表面应有克制、细腻、真实的金属车漆反光
- 亮部通透，暗部厚重，反光锐利但不过曝
- 不要赛博霓虹
- 不要玻璃拟态
- 不要过度荧光化
- 不要把绿色做成偏黄的草绿
- 不要做成普通 UI glossy surface
```

## 七、最终落地判断标准

如果做对了，页面看起来应该像：

* **阿斯顿·马丁 F1 赛车停在摄影棚里**
* 不是普通绿色科技网站
* 不是新能源品牌官网
* 不是霓虹赛博风
* 而是 **深色、精密、克制、贵、快，并且有真实车身金属漆光泽**

如果做错了，最常见的观感会变成：

* 绿色太轻
* 绿色太艳
* Lime 太多
* 没有体积感
* 没有车漆反光
* 更像互联网后台，而不是赛车品牌视觉
